# Quake API
#### Aplicação feita em NodeJS (Typescript) responsável por ler os logs do Game Quake e expor os dados em API.

</br>

### Dependências:
* NodeJS
* Typescript
* Express
* Cors
* Jest
* Eslint
* Prettier

#
### Rotas da aplicação:
 - [Coletar Games](http://localhost:3000/api/games)

 
#
### Configuração das variáveis de ambiente:
 - [Environment](.env)


### Instalação das dependências:
```bash
$ npm install
```

### Comandos para inicializar a aplicação:
```bash
# Buildar aplicação
$ npm run build

# Modo desenvolvimento
$ npm run start:development
```

### Comandos para inicializar os testes:
```bash
# Testes unitários
$ npm run test

# Cobertura de testes
$ npm run test:coverage
```

### Comandos para inicializar o Eslint:
```bash
# Eslint
$ npm run lint

# Eslint - Ajustar
$ npm run lint:fix
```
